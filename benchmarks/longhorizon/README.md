# Long-Horizon Benchmarks

Use this suite for broader multi-file, milestone-shaped tasks that remain inspectable and intentionally runnable outside the default CI path.

- Keep longhorizon grounded in realistic repo work, not generic synthetic chores.
- Prefer cases that exercise sequencing, artifact packaging, or broader docs/refactor flows.
- Keep the tasks fixture-backed and versioned even when they are more expensive than `smoke` or `fresh`.
